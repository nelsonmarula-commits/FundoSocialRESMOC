# Fundo Social Resiliência Android WebView

Aplicativo Android WebView para abrir:

https://fundosocial.resilience.co.mz

## Como gerar APK no GitHub

1. Faça upload de todo o conteúdo deste projeto para um repositório GitHub.
2. Vá em **Actions**.
3. Abra **Build Android APK**.
4. Aguarde a execução terminar.
5. Baixe o artefato **app-debug**.

Este projeto evita o Gradle Wrapper e usa Gradle oficial via GitHub Actions.
